MediQ – Smart Hospital Management & Doctor Appointment System
MediQ is a smart, modern hospital management platform designed to simplify the patient–doctor interaction process. 
It provides real-time bed availability, doctor appointment booking  and complete appointment management — all in one place.
Built for efficiency, scalability, and a seamless user experience.

--FEATURES
1.Hospital information dashboard
2.Real-time bed availability tracking
3.Slot-based appointment scheduling
